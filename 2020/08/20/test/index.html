使用hexo+GitHub搭建博客

搭建过程可参考此视频

1.申请GitHub账号

2.创建博客仓库

3.安装nodejs

4.安装git

5.安装hexo并搭建博客框架

npm install -g hexo-cli //全局安装hexo
hexo init //初始化博客框架
npm install //安装组件
hexo g //编译框架
hexo s //本地预览
hexo d //推送框架
hexo clean //清理框架
npm install https://github.com/7ym0n/hexo-asset-image --save //使图片可以正常加载

6.修改_config.yml,关联GitHub

post_asset_folder: true
deploy:
    type: git  
    repo: https://github.com/sunwei2520/sunwei2520.github.io.git  
    branch:master
